# ETHICS Metal Forming Machineries India - Workspace

Welcome to the official repository for **ETHICS Metal Forming Machineries India** (Engineers Hydro Mech). This repository contains the Next.js web application, brochures, corporate brand assets, and product catalog resources.

## 📂 Repository Structure

* **[hydromech-website](file:///d:/projects/hydromech%20int/hydromech-website)**: The primary web application built with Next.js 14, React 18, Tailwind CSS, and TypeScript.
* **[Hydromech](file:///d:/projects/hydromech%20int/Hydromech)**: Directory containing categorized high-resolution product media:
  * Clinching & Insertion machinery
  * Busbar Processing lines
  * Profile Section Bending machinery
  * Plate Rolling machinery
* **[Hydromech Final Brochure.pdf.pdf](file:///d:/projects/hydromech%20int/Hydromech%20Final%20Brochure.pdf.pdf)**: Official product brochure for corporate distribution.
* **[brand-kit.html](file:///d:/projects/hydromech%20int/brand-kit.html)**: Clean HTML page for brand style references.
* **[logo.png](file:///d:/projects/hydromech%20int/logo.png)** / **[ETHICS METAL 2026 LOGO.png](file:///d:/projects/hydromech%20int/ETHICS%20METAL%202026%20LOGO.png)**: High-resolution corporate branding assets.

## 🚀 Getting Started

To launch the web application locally:

1. Navigate to the website subdirectory:
   ```bash
   cd hydromech-website
   ```
2. Install the workspace dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Access the site locally at [http://localhost:3000](http://localhost:3000).

For detailed information on app features, page structures, or build optimizations, check the **[hydromech-website/README.md](file:///d:/projects/hydromech%20int/hydromech-website/README.md)**.
